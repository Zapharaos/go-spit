# User Guide

This guide covers the go-spit data model and every export format in depth. Start with the data
model, then jump to the format(s) and features you need.

## Core concepts

<div class="grid cards" markdown>

- :material-table: **[Tables, Data & Columns](tables-and-columns.md)**

    The data model: rows, column definitions, hierarchies and lookups. Read this first.

- :material-palette: **[Styling, Borders & Merging](styling.md)**

    Fonts, colors, alignment, borders and cell merging shared across formats.

- :material-file-cog: **[File Options](file-options.md)**

    Control where and how files are written: paths, overwrite, compression, temporary files.

- :material-math-log: **[Logging](logging.md)**

    Plug go-spit into your existing logger.

</div>

## Export formats

<div class="grid cards" markdown>

- :material-file-delimited: **[CSV Export](csv-export.md)**

    Simple tabular data with custom delimiters and multi-level headers.

- :material-file-excel: **[XLSX Export](xlsx-export.md)**

    Advanced spreadsheets with styling, borders, merging and hierarchical headers.

- :material-language-html5: **[HTML Export](html-export.md)**

    Styled `<table>` output and full composed documents (headings, lists, sections).

- :material-file-pdf-box: **[Generating a PDF](pdf-export.md)**

    Produce print-ready HTML and render it to PDF with the engine of your choice.

- :material-google-spreadsheet: **[Google Sheets](google-sheets.md)**

    Push tables directly to Google Sheets with your own authenticated client.

</div>
